== Description ==

Enable new shortcake for create container that includes all info tweak Cydia of the your article, such as name tweak and cost and link for download by repo of the developer.
Into container is show one button for access directly to repo of the developer. On click this button, if you have on iPhone jailbroken, open Cydia app.
The button use  3rd party as a service, is important that into article is specify the repository cydia correct. 

I suggest you see the screenshots.

== Installation ==

Upload the Akismet plugin to your blog, Activate it.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/shortcode-cydia` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
